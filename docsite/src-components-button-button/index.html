Button 组件

使用方式如下所示：

() => {
  const btnData = {
    func: {
      btnType: 'half',
      btnWord: '立即开通',
    },
    style: {
      btnWidth: '180',
      btnHeight: '100',
      btnBgColor: 'linear-gradient(270deg,#ffc900,#ffd841)',
      btnRadius: '50',
    },
    data: {
      event: {
        type: 'out',
        filter: '() => {}',
      },
      getData: {
        currPrice: '98',
        oldPrice: '199',
      },
    },
  }
  return (
    <Button {...btnData.func} {...btnData.style} {...btnData.data}>
      我是按钮1
    </Button>
  )
}

基本属性

属性名称说明默认值
btnType按钮类型--
btnWord按钮文案--
btnWidth按钮宽度--
btnHeight按钮高度--
btnBgColor按钮颜色--
